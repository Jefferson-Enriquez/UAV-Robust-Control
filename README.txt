# Robust Control Using a H∞ Mixed Sensitivity Approach for a UAV with a Suspended Payload  
**ID**: 9285  

## 1. Description:  
- **Lineal5simu.m**: contains the script for the state-variable function with the UAV parameters.  
- **LinealYcontrol5.m**: contains the script for the linearization of the original system, the creation of the extended plant, and the control.  
- **Weighting5**: contains the script to build the weighting functions.  
- **Graficas.m**: contains the script to display the graphs of the “Weighting5” functions.  
- **PlantaDrone5.slx**: contains the Simulink file of the original system described in “Lineal5simu.m.”  
- **Prueba5.slx**: contains the Simulink file for applying control to the original system and the linearized system.
